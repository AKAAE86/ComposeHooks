需要注意：Debounce 不返回计算结果，在 Compose 中我们无法使用 Debounce 透传出结算结果，应该使用状态，而非 Debounce 的返回值。 例如我们有一个计算函数，我们应该设置一个状态作为结果的保存。函数计算后的结果，通过调用对应的 setState(state:T) 函数来传递。保证结算结果（状态）与计算解耦。 这样我们的Debounce 就可以无缝接入。
